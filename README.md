##  📦 Installation

```jsx
npm install --save zfg
```

## Chart

- 饼形图 => PIE
- 柱状图 => COLUMN, INTERVAL
- 散点图 => POINT
- 线状图 => LINE
- 面积图 => AREA
- 中国地图 => CHINAMAP

## Usage

```js
import { LINE } from 'zfg';

<LINE data={data} config={config} />
```
